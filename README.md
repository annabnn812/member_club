# member_club
## Getting Started
  git clone https://github.com/annabnn812/member_club.git

  cd member_club/frontend 

  ```bash
npm i
# or
yarn install
```
  
First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
